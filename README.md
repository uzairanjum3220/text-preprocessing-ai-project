 Text Data Preprocessing for AI

Programming for Artificial Intelligence – Course Project

## Project Overview
This project demonstrates basic text data preprocessing techniques used in Artificial Intelligence systems. Raw SMS text data is cleaned and transformed into a structured format suitable for AI applications.

Dataset
The SMS Spam Collection dataset is used, which contains labeled SMS messages classified as spam or ham.

 Preprocessing Steps
- Conversion of text to lowercase
- Removal of punctuation
- Removal of numbers
- Removal of stop words
- Tokenization

Project Structure

Tools Used
- Python
- Pandas
- NLTK
- Visual Studio Code
- GitHub

How to Run
1. Install dependencies:
   pip install pandas nltk
2. Navigate to preprocessing folder:
   cd preprocessing
3. Run the script:
   python preprocess.py

 Output
A cleaned dataset ready for AI and Machine Learning tasks.
